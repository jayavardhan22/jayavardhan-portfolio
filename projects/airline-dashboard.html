Airline Operations Dashboard

This project focuses on analysing airline operational performance using interactive dashboards.

Objective

To build an analytics dashboard that provides insights into airline operations including delays, flight performance and operational trends.

Tools Used

  • Power BI
  • Tableau
  • SQL
  • Data Visualization

Key Features

  • Flight delay analysis
  • Operational performance monitoring
  • Interactive dashboard filters
  • Data-driven insights

These dashboards help decision makers quickly understand operational bottlenecks and improve airline efficiency.
