Machine Learning on Bank Marketing Dataset

This project focuses on predicting whether a customer will subscribe to a bank term deposit using machine learning classification models.

Dataset

The dataset contains customer information including age, job, education, account balance, contact history and previous campaigns.

Tools Used

  • Python
  • Pandas
  • Scikit-learn
  • Machine Learning

Models Implemented

  • Logistic Regression
  • Decision Tree
  • Random Forest

Outcome

The project compares multiple classification models to determine which algorithm performs best for predicting customer subscription.
