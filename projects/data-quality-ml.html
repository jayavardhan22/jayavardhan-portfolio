Data Quality Validation using Machine Learning

This research project focuses on improving data quality in traffic collision datasets using machine learning and rule-based validation.

Objective

To identify inconsistencies, missing values and anomalies in large-scale traffic accident datasets.

Approach

  • Machine Learning anomaly detection
  • Rule-based validation system
  • Data preprocessing and cleaning

Tools Used

  • Python
  • Pandas
  • Scikit-learn
  • Data Quality Validation Techniques

Impact

Improving data quality ensures more reliable insights for traffic safety analysis and policy making.
